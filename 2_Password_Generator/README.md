# 🔐 Random Password Generator

This is a simple Random Password Generator built using HTML, CSS, and JavaScript.

The project generates a random password containing uppercase letters, lowercase letters, numbers, and special characters. Users can generate a new password with a single click and copy it to the clipboard for easy use.

## Features

* Generate random passwords instantly
* Uses letters, numbers, and symbols
* Copy generated password with one click
* Simple and user-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

1. Click the **Generate Password** button.
2. A random password is generated and displayed in the input field.
3. Click the **Copy** button to copy the password to the clipboard.
