# 📝 To-Do List App

A simple and responsive **To-Do List Application** built using **HTML, CSS, and JavaScript**.

## 🚀 Features

### ➕ Add Tasks
- User can type a task in the input field.
- Clicking the **Add** button adds the task to the list.
- The input field gets cleared after adding the task.

### ✅ Mark Task as Completed
- Clicking on a task marks it as completed.
- Completed tasks show:
  - Strikethrough text

### ↩️ Unmark Task
- Clicking on a completed task again removes the completed state.
- The task becomes active again.

### ❌ Delete Task
- Each task contains a **cross (×) icon** on the right side.
- Clicking the cross icon removes the task from the list.

### 💾 Local Storage Support
- All tasks are stored in the browser's **Local Storage**.
- Tasks remain saved even after:
  - Refreshing the page
  - Closing and reopening the browser
- Deleted tasks are permanently removed from storage.

## 🛠️ Technologies Used

- HTML5
- CSS
- JavaScript
- Local Storage API
